Aim:- Implement 8-queens problem using backtracking approach.

                
Program 1:

#include <stdio.h>
#include <stdbool.h>

#define N 8

void printSolution(int board[N][N]) {
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < N; j++)
            printf(" %d ", board[i][j]);
        printf("\n");
    }
}

bool isSafe(int board[N][N], int row, int col) {
    int i, j;
    for (i = 0; i < col; i++)
        if (board[row][i])
            return false;
    for (i = row, j = col; i >= 0 && j >= 0; i--, j--)
        if (board[i][j])
            return false;
    for (i = row, j = col; j >= 0 && i < N; i++, j--)
        if (board[i][j])
            return false;
    return true;
}

bool solveNQueensUtil(int board[N][N], int col) {
    if (col >= N)
        return true;
    for (int i = 0; i < N; i++) {
        if (isSafe(board, i, col)) {
            board[i][col] = 1;
            if (solveNQueensUtil(board, col + 1))
                return true;
            board[i][col] = 0;
        }
    }
    return false;
}

bool solveNQueens() {
    int board[N][N] = { {0} };
    if (solveNQueensUtil(board, 0) == false) {
        printf("Solution does not exist\n");
        return false;
    }
    printSolution(board);
    return true;
}

int main() {
    solveNQueens();
    return 0;
}


            
                
Algorithm :

1. Start
2. Define a function printSolution to print the chessboard configuration.
3. Define a function isSafe to check if it's safe to place a queen in the current position.
4. Define a recursive function solveNQueensUtil to solve the N-Queens problem using backtracking:
   a. Base case: If all queens are placed (column index col reaches N), return true.
   b. Try placing a queen in each row of the current column (col):
      i. Check if it's safe to place a queen in this position using the isSafe function.
      ii. If safe, place the queen in the current position and recur to place the rest of the queens in the next column (col + 1).
      iii. If placing the queen doesn't lead to a solution, backtrack by removing the queen from the current position.
   c. If no row is safe to place the queen in the current column, return false.
5. Define a function solveNQueens to initialize the chessboard, call the solveNQueensUtil function, and print the solution.
6. In the main function, call the solveNQueens function to solve the N-Queens problem.
